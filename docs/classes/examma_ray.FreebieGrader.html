    Computes the number of points earned for the given graded question. Heads up! This could be negative or more than the number of points a question is worth, depending on the type of grader. That's not the concern of the grader, and it's your decision on how to handle it elsewhere (e.g. by clamping the value between 0 and max points possible on a question).
